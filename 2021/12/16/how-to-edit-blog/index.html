如何在本博客发布文章

2021-12-162022-04-30

在博客发布文章是什么意思？小伙靠它月赚 500w！
众所周知，在此博客发布文章是最近非常火的一个词。那么在此博客发布文章到底是什么梗呢？相信大家对在此博客发布文章都很熟悉，在此博客发布文章就是我们每天都会经常遇到的，但是在此博客发布文章是怎么回事呢？就由小编来为大家介绍一下。
😏 (doge)

咳咳, 此篇文章旨在教会新手如何使用 Hexo 在本博客发表文章
(有误还请大佬指正)

准备工作

“工欲善其事，必先利其器”
让我们得到必要的工具, 再开始你的 月赚 500w 发表文章之路

前期准备

你需要以下东西:

  • Node.js: 配置教程
  • Git: 使用教程
  • Github 账户
  • Markdown 知识: 教程
  • Visual Studio Code (VS Code)
  • 你聪明的脑袋

配置好以上东西以后, 加入到我们的 Github 组织 Shade-Man
联系 stars_sea@outlook.com
(因为文章源文本只有组织成员才能看见)

注意

Node.js v14 需要 Windows 8 以上环境, Windows 7 的同学可以下载 v12
LTS (长期支持版本) 在 这里 下载

Github 进不去正常, 建议科学上网, 多刷亿遍

VS Code 建议下载 System Installer (如果环境是 Windows)

环境配置

  1. 安装 hexo-cli 库

  2. 找一个文件夹, 克隆 Shade-Man/HexoBlog 仓库
    (任选一条)

    完成后你会看见 done 这个字眼, 反之多试几次 (实在不行科学上网)

  3. 进入克隆的文件夹, 执行这条命令, 下载必要库
    (任选一条, 第二条使用淘宝镜像下载)

施展才华

经过以上步骤, 你的编写环境准备就绪
还差一步, 你就可以在此博客施展你的才华了

准备编写

  1. 在你克隆的仓库 (文件夹) 打开 VS Code

  2. 找到 VS Code 顶上的 终端 / Terminal
    新建一个终端 (新建终端 / New Terminal)

  3. 在终端内输入 hexo new "文章名称"
    如果成功, 你会看见这样的输出

  4. 之后, 你就可在 D:\Repos\HexoBlog\source\_posts\文章名称.md
    利用 Markdown 写新的文章 (VS Code 右上角可打开预览)

预览, 发布!

在写完后, 请务必记得保存

在终端输入 hexo s 可在本机打开服务器
在浏览器输入给定地址, 即可预览博客

比如 (可能会有报错, 一般不用怎么管):

在浏览器输入 http://localhost:4000 即可预览
在终端按下 Ctrl + C 即可关闭服务器

确认无误后, 可输入 hexo d 将博客上传到 Github Pages
等待一会, 你就可在 https://shade-man.github.io 看到你的文章了

Hexo 命令选项

  • clean: 清除缓存文件之类的
  • deploy / d: 将你的网站部署到你的远程仓库
  • generate / g: 生成网站静态文件
  • new / n: 创建新的文章
  • server / s: 开启 Hexo 服务器

建议: 在执行 hexo d|g|s 时请将命令替换成 hexo clean && hexo d|g|s
