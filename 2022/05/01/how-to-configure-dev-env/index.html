如何配置基础开发环境

2022-05-012022-05-01

此处所指的 “基础开发环境” 主要包括:

  • 语言运行时 & 编译器
  • 好用的文本编辑器

并不一定要全部更着教程走, 可以按需执行

Python 环境

Python 安装 (Standard)

Python 是动态语言, 实时解释, 实时运行
所以一个 Python 安装包包含运行时和解释器 (编译器)

可以从 Python.org 下载其环境

但是我更建议从国内镜像站下载 (更快):

  • 华为: https://mirrors.huaweicloud.com/python/
  • 淘宝: https://npm.taobao.org/mirrors/python/
    会重定向到 https://registry.npmmirror.com/binary.html?path=python/

Note:

  1. 页面拉到底的版本不一定是最新版本, 比如 3.10.0 就排在 3.2 前面
    (切记一定要下 Python3.x)
  2. 下载请看准安装包后缀 (根据你的系统要求, 一般为 .msi/.exe, 64 位选 amd64)
    主版本后跟 rc1 之类字符的是测试版, 不建议下载
    Python3.9.0 及其后版本不支持 Windows7 及其以前版本
  3. 安装时请勾选 “Add Python 3.x to PATH” （不这样之后很麻烦）
    没有 VS 就不要勾选 “Download debug binaries”
  4. 这里的镜像站不是 pypi 的, 而是 Python 安装包的, 请勿混淆

Python 自带代码编辑器 (IDLE)
当然, 你也可以选择 VS Code 或其他编辑器

Python 配置

一般不用怎么配置, 如果想要使用第三方库又怕下载慢, 可以这样配置

镜像地址 (用以下地址替换, 包括尖括号):

  • 豆瓣: http://pypi.doubanio.com/simple
  • 网易: https://mirrors.163.com/pypi/simple
  • 清华: https://pypi.tuna.tsinghua.edu.cn/simple
  • 阿里云: https://mirrors.aliyun.com/pypi/simple
  • 腾讯云: https://mirrors.cloud.tencent.com/pypi/simple

有的源可能要设置 trusted-host, 根据安装包的报错提示来吧, 这里不做过多赘述

Anaconda 安装 (Optional)

Anaconda 相当于一个 Python 发行版,
内含许多科学运算、数据分析的库 (故比较大) (这个是个人见解)

一般没啥需要可以不装, 可能装了也玩不来…
(但是可以装 13 啊)

Java 环境

Java 不同于 Python, JRE (运行时) 和 JDK (开发工具包) 是分开的 (Java8 及其以前是如此)
但是安装 JDK 就没必要装 JRE 了, 因为里面的 java.exe 和 JDK 里的没啥差别

我所熟知的发行版有 OpenJDK, OracleJDK

OpenJDK (Recommended)

开源的 Java 环境

可以从下面连接获取压缩包, 解压即可, 不用安装
https://jdk.java.net/archive/

新版 Minecraft 开发使用 Java16 (mc1.17) 或 Java17 (mc1.18)
旧版使用 Java8

OracleJDK

Oracle 的 Java 环境, 部分版本用来商用可能会被追究法律责任

可从下面获取安装包 (请注意是否为 Java SE Development Kit (JDK))
https://www.oracle.com/java/technologies/downloads/

安装 JDK8 时建议取消勾选 Java Runtime Environment (JRE)

环境配置

  1. 新建一个环境变量 JAVA_HOME 指定为你解压/安装 JDK 的目录
    (如 C:\Program Files/Java/jdk17)
  2. 在 PATH 环境变量后加入 %JAVA_HOME%\bin
  3. 验证: 在终端中输入 java -version
    应会出现和下面相同的东西:

不清楚可以参考这个

Java 没有自带的文本编辑器, 推荐使用 Idea 或 VS Code

.NET (C#)

和 Java 一样, 分为 Runtime (运行时) 和 SDK (开发工具包)

它也有许多版本, 据我所知, 早期分为 .NET Framework 和 .NET Core
之后两个版本被合并为 .NET

可以从下面获取安装包, 但更建议从 VS 中安装
如果嫌 VS 大, 坚持用 VS Code, 那么也不阻拦

.NET Framework

下载地址:

  • 官网: https://dotnet.microsoft.com/zh-cn/download/dotnet-framework
  • 阿里云分享: https://www.aliyundrive.com/s/bnVSdP9RTBP 提取码: 6sd5

.NET Core

下载地址:

  • 官网: https://dotnet.microsoft.com/en-us/download/dotnet
  • 阿里云分享: https://www.aliyundrive.com/s/bnVSdP9RTBP 提取码: 6sd5
    (对应文件名 dotnet-sdk-3.1.418-win-x64.exe)

.NET

下载地址:

  • 官网: https://dotnet.microsoft.com/en-us/download
  • 阿里云分享: https://www.aliyundrive.com/s/bnVSdP9RTBP 提取码: 6sd5
    (对应文件名 dotnet-sdk-6.0.202-win-x64.exe)

C/C++ (C艹)

可以使用下面两种方法进行 C/C++ 开发

Visual Studio (Recommended)

从下面链接安装 Visual Studio Community 后,
选择使用 C++ 的桌面开发, 确定等待安装完成即可
https://visualstudio.microsoft.com/zh-hans/vs/

MinGW + Visual Studio Code

MinGW

可参考 http://c.biancheng.net/view/8077.html

可从下面链接下载对应的版本 (一般选择 x86_64-win32-seh):
(不知道下载啥? 参考这个)
https://sourceforge.net/projects/mingw-w64/files/

下载慢? 试试迅雷?
或者在这里下载
提取码: 6sd5

下载后解压, 将 bin 目录加入 PATH 环境变量 (参考 Java)

VS Code

可从这里下载:
https://code.visualstudio.com/#alt-downloads

建议下载 System Installer (如果你是 Windows)

之后安装对应的扩展(包)就行

然后根据弹出窗口配置
